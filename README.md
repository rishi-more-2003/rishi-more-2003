# Hi, I'm Rishi 👋

I'm a Computer Science master's student at **Johns Hopkins University**, graduating in **May 2026**.  
My work focuses on **ML systems**, **efficient LLM inference**, **retrieval-augmented generation**, and **compute-aware reasoning**.  
I'm a co-first author of an **ICML 2026** paper on risk-controlled test-time compute for reasoning LLMs.  
I'm especially interested in building scalable AI infrastructure for modern ML and LLM workloads.

## What I Work On

- **Efficient LLM inference** and test-time compute optimization
- **Risk-controlled reasoning** for large language models
- **Temporal-aware retrieval-augmented generation**
- **Post-training evaluation** and preference optimization
- **Scalable ML infrastructure** using Python, PyTorch, TensorFlow, Docker, Kubernetes, and AWS

## Featured Research

### Conformal Thinking: Risk Control for Reasoning on a Compute Budget
- **Accepted at ICML 2026**
- Research on adaptive test-time compute for reasoning LLMs under compute budgets
- Focuses on controlling risk while reducing unnecessary reasoning cost
- [arXiv](https://arxiv.org/abs/2602.03814)

### Temporal-Aware Retrieval-Augmented Generation
- Built a RAG pipeline over 10K+ non-stationary documents
- Used time-decay retrieval weighting and metadata-aware indexing
- Improved retrieval precision over dense-retrieval baselines in volatile knowledge domains

### Moderator-Aligned Toxicity Detection via Direct Preference Optimization
- Fine-tuned a toxicity classifier using preference signals from community feedback
- Applied Direct Preference Optimization to improve alignment with moderator labels
- Achieved strong generalization on context-dependent toxicity cases
