# Hi, I'm Rishi 👋

ML Research Engineer focused on **post-training evaluation**, **synthetic data pipelines**, and **production inference systems**. I work at the intersection of reasoning LLMs, retrieval systems, and scalable ML infrastructure.

I’m currently pursuing an **M.S.E. in Computer Science at Johns Hopkins University** and have worked on:
- **Distribution-free risk control for reasoning LLMs** under compute budgets
- **Temporal-aware RAG pipelines** over non-stationary corpora
- **Automated data quality and hallucination detection** for retrieval systems
- **GAN-based privacy-preserving vision systems** optimized for real-time inference

## What I’m working on

- **Evaluation frameworks** for compute-quality tradeoffs
- **Scalable MLOps systems** for training and inference
